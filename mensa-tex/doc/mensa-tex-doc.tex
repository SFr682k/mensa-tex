	\section{Using the class}
	Using this class is as easy as using the \mintinline{LaTeX}{\documentclass{mensa-tex}} command.
	
	\bigskip
	Following options are available:
	\begin{itemize}
		\item[\texttt{app}] Use a layout optimized for small screens using A6--paper
		\item[\texttt{en-GB}] Use an English localisation, British variant \textit{(default)}
		\item[\texttt{en-US}] Use an English localisation, American variant
		\item[\texttt{german}] Use a German localisation
	\end{itemize}
